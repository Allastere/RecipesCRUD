# Recipes CRUD API

Simple CRUD REST API built with **FastAPI**, **SQLAlchemy**, and **SQLite**.

## Features
- Create / Read / Update / Delete recipes
- SQLite database
- SQLAlchemy ORM
- Automatic OpenAPI docs (Swagger)
- Clean project structure

## Tech Stack
- Python 3.11
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn
